# 常用快捷键

## 创建

```
ctrl + d  复制上一行代码
alt + insert  鼠标在文件夹上，可创建子路径或类
alt + insert  鼠标在代码区，可插入构造和getter方法
ctrl + alt + v  生成变量名，如生成new Student()的引用变量名
```

接口相关

```
生成实现类
alt + enter 新创建的接口，用鼠标点击接口名字，同时按住Alt+Enter键，可生成实现类
重写方法
alt + enter 点击实现类名，按住ALT+ENTER键，可implement methods
ctrl + i    鼠标放在接口方法上，可实现接口方法
```

## 修改

```
ctrl + alt + t    快速添加try-catch
alt + 左键         可以按列复制或删除
ctrl + shift + U  将选中代码大小写转换
Ctrl + O          重写父类方法
shift + F6        快捷重命名，会修改所有关联地方
alt + enter       放在成员变量上，可修改访问权限
```



## 查找

```
ctrl + tab      当前文件与上一文件来回切换
ctrl + h        查看当前接口的实现类或者类的所有子类 
ctrl + alt + h  查看方法的调用关系，鼠标放在方法上
ctrl + shift +f 项目中全局搜索
ctrl + N        全局搜索所有的类等 
shfit + shfit    连按2次shift可以搜索项目中所有东西
ctrl + alt + B  鼠标放在方法上，可以跳到方法的实现处
ctrl + alt + U  可查看类的继承diagrams
ctrl + F12      查看类的所有成员变量和方法
ctrl + shift + F12 代码页面最大化 
```



## 格式化

```
ctrl + alt + i  优化鼠标所在行代码缩进
ctrl + alt + L  一键格式化代码，也可以只对选中的代码格式化
ctrl + alt + O  去除.java文件前的多余import导入
```



## 执行

```
ctrl + shift + F10  执行光标所在的方法，常用在单元测试类中

```



## 重构

```
ctrl + alt + M  对选中区域抽取方法
ctrl + alt +shift +T  选中method可抽取方法，没ctrl+alt+M方便
ctrl + alt +shift +T  选中方法参数，通过Parameter Object抽取参数为对象
alt + del 或alt+enter  删除方法未使用的形参
```
